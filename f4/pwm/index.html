Pulse Width Modulation

You can use the Pwm interface with these TIM instances

TIM1

  • CH1 = PA8
  • CH2 = PA9
  • CH3 = PA10
  • CH4 = PA11

TIM2

  • CH1 = PA0
  • CH2 = PA1
  • CH3 = PB10
  • CH4 = PA3 (Unimplemented: conflicts with USB USART2_RX)

TIM3

  • CH1 = PA6
  • CH2 = PC7
  • CH3 = PB0
  • CH4 = PB1

TIM4

  • CH1 = PB6
  • CH2 = PB7
  • CH3 = PB8
  • CH4 = PB9
